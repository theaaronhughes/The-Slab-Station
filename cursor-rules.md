\# Cursor Rules for The Slab Station

\- This is a static site. Do not add frameworks or bundlers.

\- Only modify: index.html, script.js, style.css (unless explicitly asked).

\- Keep Tailwind classes; do not reformat into different CSS frameworks.

\- Do NOT change serverless functions or package.json unless asked.

\- Avoid adding external JS libraries. Prefer small CSS/JS adjustments.

\- Every change must preserve existing behavior and visual style.

\- When editing, explain what changed and why in the PR description.
- Explain changes at the end of the message (files + why).
